# online-exam-monitoring-integrity-analytics-platform
Online Exam Monitoring &amp; Integrity Analytics Platform


A Flask-based online examination monitoring platform developed as part of the Infosys Springboard Internship.

## Tech Stack
- Python
- Flask
- SQLite
- OpenCV
- Faker

## Project Structure

- app.py - Main Flask application
- database/ - Database schema and initialization
- routes/ - Flask route handlers
- modules/ - Reusable modules
- templates/ - HTML templates
- static/ - CSS, JavaScript, and images

## Setup

1. Install dependencies:
   pip install -r requirements.txt

2. Initialize the database:
   python database/init_db.py

3. Run the application:
   python app.py
